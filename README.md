# 概要

大村研究室 学部 4 年生での研究「純正律での和音演奏を支援するシステムの構築」の開発リポジトリ

# commands

### 開発環境作成

仮想環境作成

```
python -m venv [envname]
```

パッケージインストール

```
pip install pytk
pip install librosa
pip install sounddevice
pip install matplotlib
pip install pyinstaller
```

### 開発環境立ち上げ

```
[envname]/Scripts/activate
```

### 終了

```
deactivate
```
